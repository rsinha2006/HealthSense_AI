# HealthSense AI & Smart Doctor Recommendation System

## Overview

HealthSense AI is a responsive healthcare web application designed to help users understand their symptoms through an intelligent symptom assessment system.
The application analyzes user-selected symptoms, predicts possible health conditions based on predefined logic, estimates the risk level, and recommends the most suitable medical specialist for consultation.

## Features

- Smart Symptom Checker
- AI Chatbot
- Doctor Recommendation
- Disease Probability Estimation
- Risk Level Indicator (Low / Medium / High)
- Fully Responsive Design
- Modern Healthcare UI

## Disclaimer

This project provides an educational demonstration of symptom-based health assessment.
